\doxyparagraph*{\begin{center}This is a community project and not an official Laravel one \end{center} }

Laravel Zero was created by, and is maintained by \href{https://github.com/nunomaduro}{\texttt{ Nuno Maduro}}, and is a micro-\/framework that provides an elegant starting point for your console application. It is an {\bfseries{unofficial}} and customized version of Laravel optimized for building command-\/line applications.


\begin{DoxyItemize}
\item Built on top of the \href{https://laravel.com}{\texttt{ Laravel}} components.
\item Optional installation of Laravel \href{http://laravel-zero.com/\#/?id=database}{\texttt{ Eloquent}}, Laravel \href{http://laravel-zero.com/\#/?id=log}{\texttt{ Logging}} and many others.
\item Supports interactive \href{http://laravel-zero.com/\#/?id=interactive-menus}{\texttt{ menus}} and \href{http://laravel-zero.com/\#/?id=desktop-notifications}{\texttt{ desktop notifications}} on Linux, Windows \& Mac\+OS.
\item Ships with a \href{http://laravel-zero.com/\#/?id=scheduler}{\texttt{ Scheduler}} and a \href{http://laravel-zero.com/\#/?id=build-a-standalone-application}{\texttt{ Standalone Compiler}}.
\item Integration with \href{https://github.com/nunomaduro/collision}{\texttt{ Collision}} -\/ Beautiful error reporting
\end{DoxyItemize}
